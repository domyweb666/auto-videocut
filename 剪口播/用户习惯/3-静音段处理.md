# 静音段处理

## 阈值规则

**使用者偏好（2026-04-15 更新，依批量訓練確認）**

| 静音长度 | 处理 |
|---------|------|
| < 1.2秒 | **保留** - 自然停頓、句間間隔、氣口，不刪 |
| ≥ 1.2秒 | **預選刪除** - 明顯卡頓或大段空白 |

> 使用者偏好保留句子間約 1 秒的自然停頓，確保語氣完整不倉促。

## 输出格式

**整段标记，不拆分**

示例：3.2 秒静音 → 输出 1 条
```
| 64-66 | 12.86-15.80 | 静音3.2s | | 删 |
```

用户在审核网页中取消不想删的。

## 刪除邊界（不倉促切段）

刪除片段時，**預留前後緩衝**，確保：
- 每句話都完整講完，不切掉尾音或氣音
- 句子之間保留自然的呼吸間隔
- 不要讓觀眾感覺句子被生硬截斷

```
❌ 刪到剛好字結束的時間點 → 聽起來倉促
✅ 多保留 0.1~0.2s 緩衝 → 自然收尾
```

## 特殊情况

### 长静音
连续 5s+ 的静音整段标记，预选删除：
```
| 323-371 | 71.38-131.38 | 静音60s | | 删 |
```

### 开头静音
视频开头的静音必删：
```
| 0 | 0.00-1.00 | 静音1s | 开头静音 | 删 |
```
