# 保留連接詞（使用者風格）

## 背景

此規則來自使用者 SRT 審稿對照分析（2026-04-13）。
使用者明確表示這些連接詞是個人表達風格的一部分，**不應刪除**。

## 必須保留的詞

以下詞語出現時，**不論上下文，一律保留**：

```
但是、不是的、事實上、也就是說、就是說、你要、
如果、那麼、所以、因此、換句話說、我們、怎麼、
比如說、例如說、舉個例子、簡單來說
```

## 判斷原則

- 這些詞是**語義連接器**，不是口誤或卡頓
- 即使它們短暫出現在靜音前後，也**不要標記刪除**
- 只刪「同一個詞連說兩次」（卡頓詞規則），不刪單次出現

## 範例

| 原文 | 處理 |
|------|------|
| 但是這其實是囚徒困境 | ✅ 保留 |
| 事實上，囉嗦才是精準 | ✅ 保留 |
| 就是說，如果每一條通訊 | ✅ 保留 |
| 就是就是，如果... | ❌ 刪「就是就是」（卡頓詞規則） |
