# 連續語氣詞

## 模式

兩個語氣詞連在一起：

```
嗯啊、啊呃、哦嗯、呃啊
```

## 檢測

```javascript
const fillerWords = ['嗯', '啊', '哎', '誒', '呃', '額', '唉', '哦', '噢', '呀', '欸'];

if (fillerWords.includes(curr) && fillerWords.includes(next)) {
  markAsError(curr, next);
}
```

## 刪除策略

全部刪除。
