# 語氣詞檢測

## 語氣詞列表

```javascript
const fillerWords = ['嗯', '啊', '哎', '誒', '呃', '額', '唉', '哦', '噢', '呀', '欸'];
```

## 刪除邊界

```
錯誤：刪語氣詞時間戳 (語氣詞.start - 語氣詞.end)
      → 可能刪掉前面字的尾音

正確：從前一個字的 end 到後一個字的 start
      → (前字.end - 後字.start)
```

## 用戶偏好

- 保留自然語氣，不要為了「乾淨」而刪語助詞
- 「那」「那個」「然後」等連接詞屬於自然語氣，**不刪**
- 「嗯」作為過渡時保留，只刪明顯卡頓（如連續「嗯嗯嗯」）
- 刪除標準是「重複」而不是「語助詞」— 只有重複說了才刪，單獨出現的語助詞保留
