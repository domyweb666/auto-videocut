# 句內重複檢測

## 定義

同一句內，短語 A 重複出現，中間夾雜 1-3 個字。

## 模式

```
A + 中間字 + A
```

## 案例

| 原文 | 模式 | 刪除 |
|------|------|------|
| 所以小所以 | 所以+小+所以 | "所以小" |
| 然後就會然後 | 然後+就會+然後 | "然後就會" |
| 任務3任務3 | 任務3+任務3 | 第一個 |
| 什麼關係什麼 | 什麼+關係+什麼 | "什麼關係" |

## 不是口誤

| 原文 | 原因 |
|------|------|
| 任務1任務2任務3 | 列舉 |
| to do | 英文 |
| 一個一個地 | 強調 |
