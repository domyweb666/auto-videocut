# 殘句檢測

## 定義

話說到一半突然停住，後面接了靜音或重新開始。

## 核心原則

**整句刪除**：識別出殘句後，從句首到句尾全部刪除，不只是刪結尾幾個字。

## 正確分析方法

```
✓ 正確：先分句 → 判斷完整性 → 整句刪除
✗ 錯誤：逐字掃描 → 發現異常結尾 → 只刪結尾
```

### 步驟

1. **先按靜音切分句子**（靜音≥0.5s作為分隔）
2. **判斷每句是否完整**（語義、語法是否自然）
3. **殘句整體標記刪除**（從 startIdx 到 endIdx）

## 模式

```
殘句(整句) + [靜音] + 完整句
    ↓
  全刪
```

## 案例

| 殘句 | 後續 | 刪除範圍 |
|------|------|---------|
| "他呢" | [靜音] + "這是我剪出來的..." | "他呢" 整句 |
| "為什麼做這個東西呢一" | [靜音] + "做這個東西的原因是" | **整句**（不只是"呢一"） |
| "分本區別就是剪映它雖然" | [靜音3s] + "眼影它是沒有學習能力的" | 整句 + 靜音 |
| "我們先打具體怎麼做呢" | "首先" | 整句 |
| "打開我們的AI" | [靜音] + "打開我們的AI然後..." | 整句（前一個不完整版本） |

## 判斷標準

1. **句子不完整**：缺少賓語、謂語或結尾不自然
2. **後接靜音**：殘句後通常有明顯停頓
3. **後有重說**：重新開始說類似內容

## 與重複句的區別

- **重複句**：兩句都完整，只是開頭相同 → 刪較短的
- **殘句**：前一句明顯不完整，被打斷了 → 刪不完整的整句

## 常見殘句特徵

- 以"呢"、"吧"、"的"等虛詞結尾但不構成完整句
- 以數字、量詞結尾但後面沒有名詞
- 句子戛然而止，語義不完整
- 話說一半被打斷，後面重新開始

## 易錯點

```
❌ 只刪"呢一"（異常結尾）
✓ 刪"為什麼做這個東西呢一"（整個殘句）
```

**切記**：殘句的問題不只是結尾，而是整句都沒說完，所以要整句刪除。
