# 靜音段處理

## 閾值規則

**使用者偏好（2026-04-15 更新，依批量訓練確認）**

| 靜音長度 | 處理 |
|---------|------|
| < 1.2秒 | **保留** - 自然停頓、句間間隔、氣口，不刪 |
| ≥ 1.2秒 | **預選刪除** - 明顯卡頓或大段空白 |

> 使用者偏好保留句子間約 1 秒的自然停頓，確保語氣完整不倉促。

## 輸出格式

**整段標記，不拆分**

示例：3.2 秒靜音 → 輸出 1 條
```
| 64-66 | 12.86-15.80 | 靜音3.2s | | 刪 |
```

用戶在審核網頁中取消不想刪的。

## 刪除邊界（不倉促切段）

刪除片段時，**預留前後緩衝**，確保：
- 每句話都完整講完，不切掉尾音或氣音
- 句子之間保留自然的呼吸間隔
- 不要讓觀眾感覺句子被生硬截斷

```
❌ 刪到剛好字結束的時間點 → 聽起來倉促
✅ 多保留 0.1~0.2s 緩衝 → 自然收尾
```

## 特殊情況

### 長靜音
連續 5s+ 的靜音整段標記，預選刪除：
```
| 323-371 | 71.38-131.38 | 靜音60s | | 刪 |
```

### 開頭靜音
視頻開頭的靜音必刪：
```
| 0 | 0.00-1.00 | 靜音1s | 開頭靜音 | 刪 |
```
